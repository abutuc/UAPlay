About Us

UAPLAY consists of a website that provides the User a catalog to search and find the perfect movie/series from Netflix.

Our goal is to share the content from a Netflix API provided by our professor. Therefore we tried to portray a catalog/portfolio website.

This website was developed as project for ITW subject in the UA Computer Science major. This subject comes as an introductory course for Web Developing, teaching the basics, such as HTML5, CSS, JS, jQuery and KO.
